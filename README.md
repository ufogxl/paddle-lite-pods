## Installation

paddle-lite-pods is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'paddle-lite-pods' ,:git => 'https://github.com/ufogxl/paddle-lite-pods'
```
